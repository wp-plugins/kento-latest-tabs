Get list of latest post, comment and most popular post on sidebar via widget.

== Description ==

jquery tabs to get list of latest, popular and recent comment with post thumbnail or gravatar image on sidebar widget.

Live Preview: http://kentothemes.com/demo/kento-latest-tabs/

Plugin Features

* jQuery Tabs.
* Unlimited color.
* Unlimited hover color.
* Most Popular Post By Comments.
* Latest Post.
* Latest Comments.
* 100% full width to fit any screen size. 
* Post author gravatar image on post list.




== Installation ==


1. Install as regular WordPress plugin.
2. Go your Pluings setting via WordPress Dashboard and activate it.
3. Then go Appearance > Widgets. will see "Kento Latest Tabs Widget" and drag this your sidebar to display on sidebar.
4. And then you will see menu "Kento Latest Tabs" on left side you can change setting to get exact match of color.
5. save when done.
6. visit you website and enjoy.
